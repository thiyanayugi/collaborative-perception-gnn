# Scripts

Utility scripts for data processing and visualization.

## Available Scripts

- `coordinate_transformation.py`: Transform coordinates between frames
- `extract_radar_data.py`: Extract radar data from ROS bags
- `extract_vicon_data.py`: Extract Vicon motion capture data
- `interactive_point_cloud_explorer.py`: Launch interactive point cloud viewer
- `interactive_vicon_explorer.py`: Launch interactive Vicon data viewer
- `multi_robot_data_synchronizer.py`: Synchronize multi-robot data
- `point_cloud_cleaner.py`: Clean and filter point clouds
- `point_cloud_label.py`: Label point cloud data
- `point_cloud_to_graph_converter.py`: Convert point clouds to graphs
- `split_frames.py`: Split dataset into train/val/test
- `trajectory_generator.py`: Generate robot trajectories
- `visualize_global_map.py`: Visualize global warehouse map
- `visualize_robot_pcl.py`: Visualize robot point clouds

## Usage

Each script can be run independently with appropriate command-line arguments.
