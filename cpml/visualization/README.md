# Visualization Module

This module provides interactive visualization tools for collaborative perception data.

## Components

- `global_map.py`: Global warehouse map visualization
- `point_cloud_explorer.py`: Interactive point cloud exploration
- `robot_pointcloud.py`: Robot-specific point cloud visualization
- `trajectory_generator.py`: Robot trajectory visualization
- `vicon_explorer.py`: Vicon motion capture data explorer

## Features

- Interactive 3D point cloud visualization
- Real-time trajectory playback
- Multi-robot visualization
- Semantic label overlays

## Usage

Launch the interactive explorer with your processed data files.
