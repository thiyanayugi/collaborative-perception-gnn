# Preprocessing Module

This module contains all data preprocessing components for the collaborative perception pipeline.

## Components

- `coordinate_transform.py`: Coordinate frame transformations
- `frame_splitter.py`: Dataset splitting utilities
- `graph_converter.py`: Point cloud to graph conversion
- `point_cloud_cleaner.py`: Point cloud filtering and cleaning
- `point_cloud_labeler.py`: Semantic labeling tools
- `radar_extraction.py`: Radar data extraction from ROS bags
- `synchronization.py`: Multi-robot data synchronization
- `vicon_extraction.py`: Vicon motion capture data extraction

## Usage

Each module can be used independently or as part of the complete preprocessing pipeline.
