# Changelog

All notable changes to the Collaborative Perception Management Layer (CPML) project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-10

### Added
- Initial release of CPML framework
- Multi-robot collaborative perception pipeline
- GNN model architectures (GATv2, ECC, GraphSAGE)
- Comprehensive preprocessing modules
- Interactive visualization tools
- Docker support
- Extensive documentation

### Features
- 5-stage data processing pipeline
- Support for temporal windows (1, 3, 5 frames)
- Multi-robot sensor fusion
- Graph-based representation learning
- Production-ready deployment tools

## [Unreleased]

### Planned
- Additional model architectures
- Real-time inference support
